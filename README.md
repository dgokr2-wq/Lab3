# StylModa v2 — MVP после CustDev и α-тестирования

Версия подготовлена для лабораторной работы по доработке информационной системы после анализа результатов α-тестирования.

## Запуск

```bash
python main.py
```

При первом запуске создается база SQLite `stylmoda_v2.db` и тестовые данные.

## Тестовые учетные записи

| Роль | Логин | Пароль |
|---|---|---|
| Администратор | `admin` | `admin123` |
| Менеджер | `manager` | `manager123` |
| Покупатель | `buyer` | `buyer123` |

## Что изменено относительно первой версии

1. Новый визуальный стиль интерфейса: светлая бежевая тема, зеленый акцент, крупный экран входа, карточка товара.
2. Добавлены фильтры каталога по категории, размеру, цвету, цене и наличию.
3. Добавлено избранное для покупателя.
4. Добавлено оформление заказа с типом получения, адресом и комментарием.
5. Менеджер может видеть заказы и менять их статусы.
6. Добавлена форма обратной связи после α-тестирования.
7. Добавлен раздел «Карта гипотез», где зафиксированы гипотезы, проблемы, решения и результат.
8. Для администратора добавлена панель показателей и управление товарами.
9. В бизнес-логике есть валидация данных, обработка ошибок и простая блокировка записей товара при редактировании.

## Структура

- `main.py` — точка входа.
- `db.py` — создание БД, таблиц и тестовых данных.
- `security.py` — хеширование и проверка паролей.
- `services.py` — бизнес-логика: пользователи, каталог, корзина, заказы, отзывы, аналитика.
- `ui.py` — Tkinter-интерфейс.
